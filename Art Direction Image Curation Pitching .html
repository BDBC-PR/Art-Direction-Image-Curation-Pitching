STEP 2: Using 2 Images in Landscape? Follow These Steps ‚ñº
‚ö†Ô∏è ONLY IF USING 2 IMAGES:
If you're only using 1 image, skip to Step 3

RESIZING TWO IMAGES:

  1. Add both images to the template
  2. Click on one of the images
  3. In the Top Right Corner you'll see 3 horizontal dots, Click that
  4. Then select Position
  5. A panel will appear on the left side
  6. Scroll to where it shows the width px and height px
If you're using 2 photos we want each photo to be half the width:
  • Make both images 350px wide (half of 700px which is what our template is)
  • Then make each image 467px height

STEP 3: Make Background Cohesive (For 2-Image Layouts) ‚ñº

The background image should be cohesive

YOU HAVE A COUPLE OPTIONS:

  • Option 1: Use rectangle background matching method (detailed below)
  • Option 2: Remove background from both images and select neutral color
OPTION 1: RECTANGLE BACKGROUND MATCHING METHOD
  1. Add a rectangle shape to the template
  2. A circle with a filled color will appear at the top
  3. Click that circle
  4. On the left panel select the rainbow circle with the plus sign in the middle of it
  5. Select the Eye Dropper Tool
  6. Using the Eye Dropper select the background of the photo that still has its background
  7. Resize the rectangle to take up half of the space
  8. Click the three horizontal dots on the top right
  9. Select Position
  10. Select Back
OPTION 2: NEUTRAL BACKGROUND
  1. Remove the background from both images
  2. Click anywhere on the background
  3. Then click on the rainbow-filled circle in the top toolbar
  4. Select a previously used background color
‚ú® WHAT THIS DOES:
This will move the rectangle to the back of the images creating a background color that matches the other photo on your template
